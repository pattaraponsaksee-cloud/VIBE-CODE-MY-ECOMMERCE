# 🧠 NEUROVIBE - Premium Brain Booster & Digital Planner Store

**NEUROVIBE** คือเว็บแอปพลิเคชันร้านค้าออนไลน์ (E-Commerce Storefront) สไตล์ Cyber-Wellness / Biotech สำหรับจำหน่ายผลิตภัณฑ์วิตามินเสริมบำรุงสมอง (Nootropics) และสมุดดิจิทัลแพลนเนอร์เพื่อเพิ่มประสิทธิภาพการจดจ่อและการทำงานขั้นสูง (Flow State)

---

## 🎨 แนวคิดการดีไซน์และธีม (Theme & Aesthetic)
ตัวเว็บได้รับการออกแบบด้วยสไตล์พรีเมียมล้ำสมัย (Premium Modern Dark Theme) 
* **โทนสีหลัก**: สีน้ำเงินห้วงอวกาศ (`#070B13`), แสงเรืองแสงสีฟ้านีออน (`#00F2FE`), และสีม่วงนีออน (`#9B51E0`)
* **ลูกเล่น**: ลูกทรงกลมเรืองแสงเป็นฉากหลังขยับช้า ๆ (Pulsing Ambient Orbs), บล็อกการ์ดสินค้าแบบโปร่งแสง (Glassmorphism), และแอนิเมชันปุ่มกดที่ลื่นไหล
* **ฟอนต์**: แฟมิลี่สมัยใหม่อย่าง *Outfit* (สำหรับหัวข้อการเน้น) และ *Inter* (สำหรับเนื้อหาทั่วไป) จาก Google Fonts

---

## 🚀 ฟีเจอร์ที่น่าสนใจ (Key Features)
1. **Interactive Hero Section**: หน้าแบนเนอร์แสดงวิสัยทัศน์ของแบรนด์พร้อมคีย์ลัดเลื่อนลงมาชมสินค้า
2. **Dynamic Catalog Filtering**: ปุ่มสลับหมวดหมู่สินค้าแยกประเภท วิตามินเสริม (Physical) และ แพลนเนอร์ (Digital Download) ได้ทันทีแบบไม่รีโหลดหน้าเว็บ
3. **Slide-out Cart Panel**: ตะกร้าสินค้าด้านข้างแบบสไลด์คำนวณราคาสินค้ารวม, ภาษี, และค่าจัดส่งอิงจากประเภทสินค้า (เช่น ฟรีค่าจัดส่งเมื่อสั่งเฉพาะไฟล์ดาวน์โหลด)
4. **Conditional Checkout Form**: หน้าต่างชำระเงินจำลองที่จะปรับเปลี่ยนฟิลด์กรอกข้อมูลตามสินค้าในตะกร้า (ซ่อนฟิลด์ที่อยู่จัดส่งอัตโนมัติหากมีเพียงรายการแพลนเนอร์)
5. **Simulated Payment Success & Instant Download**: เมื่อสั่งซื้อเสร็จสิ้น ระบบจะแสดงหน้าจอทำรายการสำเร็จ พร้อมส่งมอบ **ลิงก์ดาวน์โหลดสมุดแพลนเนอร์ดิจิทัล** ให้ลูกค้าดาวน์โหลดไปใช้งานต่อได้ทันที
6. **Reviews Carousel**: แถบรีวิวความประทับใจระดับ 5 ดาวจากผู้ใช้งานจริงแบบสไลด์เปลี่ยนรูปอัจฉริยะ

## 🛠️ เทคโนโลยีที่ใช้พัฒนา (Tech Stack)
* **Frontend Core**: HTML5 (โครงสร้างเว็บแบบ Semantic HTML)
* **Styling**: Vanilla CSS3 (ระบบตัวแปรโทนสีและแอนิเมชันกำหนดเอง)
* **Logic**: Vanilla JavaScript ES6+ (ควบคุมการทำงานของตะกร้าสินค้า ฟิลเตอร์ ตัวสไลด์ และโมดอลการจ่ายเงิน)
* **Icons**: [Remix Icon](https://remixicon.com/) (CDN)

## 💻 วิธีการติดตั้งและเริ่มใช้งานในเครื่อง (How to Run)
เนื่องจากระบบทำงานบนฝั่งผู้ใช้งานทั้งหมด (Client-Side) คุณสามารถเปิดดูเว็บได้หลายวิธี:

### วิธีที่ 1: ดับเบิลคลิกเปิดไฟล์โดยตรง
* เปิดโฟลเดอร์ `app` และดับเบิลคลิกไฟล์ `index.html` เพื่อดูผลการแสดงหน้าเว็บผ่านเบราว์เซอร์ได้ทันที

### วิธีที่ 2: รันผ่าน Local Server (แนะนำ)
หากต้องการให้แอปพลิเคชันรันจำลองเหมือนนำขึ้นเว็บไซต์จริง:
1. เปิด Terminal หรือ PowerShell ในโฟลเดอร์หลักของโปรเจกต์นี้
2. รันคำสั่งผ่านเครื่องมือ `npx` (ต้องติดตั้ง Node.js ไว้ในระบบ):
   ```powershell
   npx http-server VIBE-CODE-MY-ECOMMERCE/app -p 8080
   ```
3. เข้าชมเว็บที่ระบุใน Terminal: [http://localhost:8080](http://localhost:8080)
